# Book Store Analysis (SQL)

## Project Overview
This project analyzes book store data using SQL to answer key questions related to sales performance, customer spending behavior, pricing patterns, and genre-wise trends. The analysis demonstrates how SQL can be used to extract meaningful insights from a relational database.

## Dataset Overview
The dataset is organized as a relational database and includes the following tables:

- Books  
- Customers  
- Orders  

These tables capture book details, customer information, and order-level transaction data required for comprehensive sales and customer analysis.

## Project Files
- **Book Store - SQL Project Overview.pdf**  
  Project overview containing selected SQL queries and their outputs.
- **Dataset/**  
  Contains CSV files used in the analysis (`Books.csv`, `Customers.csv`, `Orders.csv`).
- **SQL Queries/**  
  Contains SQL query files categorized by complexity:
  - `basic_queries.sql`
  - `advanced_queries.sql`

## Scope of This Project
This project presents selected SQL queries and their outputs to highlight important analytical questions and results.  
The complete set of SQL queries used in this project is available in the repository as SQL (`.sql`) files.
